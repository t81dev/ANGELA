## ‚öôÔ∏è Installation & Setup

### **Option 1 ‚Äî OpenAI GPT Custom GPT (No Local Runtime Needed)**

1. Go to the GPT Creation Portal:
   [https://chat.openai.com/gpts](https://chat.openai.com/gpts) ‚Üí *Create a GPT*.
2. Upload all `*.py` files and `manifest.json` from this repo.
3. Copy `/docs/prompt.json` into the GPT‚Äôs *Instructions* field.
4. Enable **long-term memory** in GPT settings.
5. Save & deploy as ‚ÄúANGELA v4.3‚Äù.

---

### **Option 2 ‚Äî Local Development via API**

#### 1. Clone the Repo

```bash
git clone https://github.com/YOUR_USERNAME/ANGELA.git
cd ANGELA
```

#### 2. Create a Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate
```

#### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

#### 4. Set Your API Keys

Create `.env` in repo root:

```env
OPENAI_API_KEY=your_openai_api_key_here
# Optional:
GROK_API_KEY=your_grok_api_key_here
```

#### 5. Run the Orchestrator

```bash
python index.py
```
